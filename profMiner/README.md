# ⛏️ profMiner - Prif Mining Dungeon Automation

> **Fast. Reliable. Efficient.** ⚡
>
> A clean, mining script with smart routing and a simple start UI.

---

## ✨ Key Features
- **One-click rock selection** with a tiny start UI 🖱️
- **Anchor-based navigation** to keep movement consistent 🧭
- **Auto deposit** at the dungeon deposit box 🧰
- **Silver fallback** that keeps you mining during respawns 🔁

---

## 🧱 Rock Options
- **Iron**: Mines the iron rock locations. 🧱
- **Silver**: Mines silver first; when all silver locations are depleted it switches to iron until silver respawns, then returns to silver. 🥈
- **Soft Clay**: Mines the soft clay rock locations. 🧱

---

## ⚙️ How It Works
- Walks to the selected rock anchor and mines nearby configured tiles. 🚶
- When inventory is full, it walks to the deposit box and deposits all items. 📦

---

## 🚀 Quick Start
1. Start the script and pick a rock type.
2. Click Start Script in the UI.

---

## 📝 Notes
- Silver respawn handling is automatic when Silver is selected.
